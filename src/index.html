Features

Get bus arrival timings

Send the bus stop code, with or without the bus service, and receive the estimated time of arrivals. Click on details button to find out about the seat availability and type of bus.

Search for bus stops

Start an inline query to find nearby bus stops. Use inline query to search for bus stops based on location.

(NEW!) Bookmark frequent eta queries

Taking the same bus to work everyday? Favourite it using the /favourites command. Bus Eta Bot now supports managing of your favourites by renaming the query with a custom name.
